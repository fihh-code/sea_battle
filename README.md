# Морской бой

## Описание игры

Это графическая игра "Морской бой", разработанная на Python с использованием библиотеки Pygame.

Игрок может расставлять корабли на поле, стрелять по полю противника и побеждать после уничтожения всех кораблей соперника.

## Режимы игры

В игре реализованы два режима:

1. Игрок против игрока
2. Игрок против компьютера

## Уровни сложности

Для режима игры против компьютера доступны три уровня сложности:

- Лёгкий
- Средний
- Сложный

## Управление

- ЛКМ — выбрать кнопку, поставить корабль или сделать выстрел
- R — повернуть корабль при расстановке
- ESC — открыть паузу
- ESC в меню паузы — продолжить игру

## Использованные технологии

- Python 3
- Pygame
- Объектно-ориентированное программирование
- Модульная структура проекта
